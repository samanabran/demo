Key Features

Every feature is designed to enhance the daily workflow of enterprise users while maintaining strict adherence to the SGC TECH AI brand identity.

Centralized Color System

All 11 brand colors defined as SCSS variables. Zero hardcoded hex values in component styles. Every color change propagates from a single source of truth.

IBM Plex Typography

Dual-typeface system: IBM Plex Serif for headings (authority, editorial weight), IBM Plex Sans for body and interface text (clarity, technical fluency).

Gold Accent Interaction Model

Required fields highlighted with gold bottom borders. Links transition to gold on hover. Tabs use navy active state with gold hover. Consistent accent language.

Architectural Navbar

Clean white navbar with champagne border, subtle shadow, and gold-tinted hover states. No colored backgrounds — precision over decoration.

Semantic Status Colors

Emerald for success, amber for warning, wine for danger, navy for info. Each semantic color derived from the brand palette with appropriate contrast.

Enhanced List Views

Navy header stripe, alternating parchment row tinting, champagne borders, and improved column header styling for enterprise data review.

Full Odoo v19 Compatibility

No deprecated APIs. Compatible with OWL components, Bootstrap 5, and modern SCSS compilation. Clean asset pipeline with proper prepend ordering.

Frequent Questions

This theme is designed and tested exclusively for Odoo v19. It leverages the latest SCSS variable system, OWL component architecture, and Bootstrap 5 framework that ship with Odoo v19. Compatibility with earlier versions (v16, v17, v18) is not guaranteed and would require independent testing and potential modifications.

All colors are centralized in primary_variables_custom.scss as SCSS variables prefixed with $sgc-. To change any brand color, modify the corresponding variable and restart Odoo to recompile the assets. No component-level edits are needed.

Yes. The theme is compatible with both Odoo Community and Enterprise editions. It only modifies backend web assets and does not interfere with Enterprise-only features such as studio, web client enhancements, or field service modules.

The theme uses ('prepend', ...) for the primary variables file to ensure its variables take priority. This follows Odoo best practices for theme modules. However, if another theme module is installed simultaneously, variable conflicts may occur. Only one theme module should be active at a time.

Navigate to Apps, search for "SGC TECH AI Enterprise Theme", and click Uninstall. After uninstallation, clear your browser cache and restart Odoo. The default Odoo visual styling will be fully restored as the theme's SCSS assets are removed from the compilation pipeline.
