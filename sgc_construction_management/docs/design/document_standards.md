## 3. Revision Standard Matrix

| Code | Status Name | Purpose |
|------|-------------|---------|
| 0 | Initial Issue | First submission |
| A, B, C | Internal / Draft | Revisions during review |
| IFR | Issued for Review | Sent to Consultant |
| IFA | Issued for Approval | Final approval cycle |
| IFC | Issued for Construction | For Site execution |
| AB | As-Built | Final project record |
| ARCH | Archived | Superseded version |
