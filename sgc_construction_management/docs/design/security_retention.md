# Security & Retention Policy

## 1. Security Matrix

| Role | Documents | Reports | Site Diary | Financials |
|------|-----------|---------|------------|------------|
| Project Manager | Full Access | Full Access | Full Access | View Only |
| Site Engineer | Upload Only | Operational Only | Full Access | No Access |
| QS / Billing | Full Access | Commercial Only | View Only | View Only |
| Finance | Full Access | Financial Only | View Only | Full Access |
| Executive | View Only | All Dashboards | View Only | View Only |
| HSE Officer | HSE Folders | HSE Only | Site Diary Only | No Access |

## 2. Retention Policy (Document Storage)

* **Project Duration:** All documents kept active.
* **Post-Completion (Defects Liability Period):** Documents archived but accessible.
* **Final Archive:** 10 years (Standard UAE practice for engineering/contracting liability).
* **Storage Location:** Odoo File Store (Internal).
